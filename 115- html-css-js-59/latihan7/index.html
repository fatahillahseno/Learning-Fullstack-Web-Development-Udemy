FAQ Sederhana

Apa itu Progressive Enhancement?

▼

Filosofi desain web yang fokus pada konten inti terlebih dahulu (HTML murni), kemudian secara bertahap menambahkan lapisan tampilan (CSS) dan interaksi (JavaScript).

Mengapa kami menggunakan Radio Button?

▼

Radio Button adalah elemen semantik yang tepat untuk pilihan tunggal, memastikan aksesibilitas dan pengiriman data formulir yang andal.

Apakah JavaScript wajib untuk interaksi ini?

▼

Ya, JavaScript diperlukan untuk menambahkan/menghapus kelas CSS secara dinamis saat ada event klik, yang tidak bisa dilakukan oleh CSS murni dalam skenario ini.
